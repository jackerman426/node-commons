This function upload a Base64 encoded file in AWS S3

Parameters
options (object) The options of the upload
Name Description
options.bucketName string The name of the bucket you want to upload the file

options.size.height number The Base 64 encoded file desired height
options.size.width number The Base 64 encoded file desired width
